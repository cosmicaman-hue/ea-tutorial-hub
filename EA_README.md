## ğŸš¨ Troubleshooting

### Issue: Can't open the system
**Solution:** 
- Check URL: http://127.0.0.1:5000/scoreboard/offline
- Verify server is running (Flask should be active)

**Solution:**
- Check browser allows localStorage
- Try incognito mode
- Check browser storage not full
- Restart browser

- Verify student exists in Students tab
- Try refreshing page
- Check date format (YYYY-MM-DD)
