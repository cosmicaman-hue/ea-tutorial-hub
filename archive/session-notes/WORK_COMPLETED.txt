                        API ENDPOINTS ADDED (8)
================================================================================

VETO Endpoints:
  ✅ POST /api/veto/restore
     - Restore VETOs (admin only)
     - Includes audit trail
     - Validates restoration amount

  ✅ GET /api/veto/verify-consistency
     - Verify veto_tracking and students[] are in sync
     - Admin only
     - Lists any inconsistencies found

Star Endpoints:
  ✅ POST /api/stars/validate
     - Validate star entry before accepting
     - Checks daily/monthly limits
     - Prevents exceeding available stars

  ✅ GET /api/stars/available/<student_id>/<month_key>
     - Get available stars for a student
     - Uses unified calculation formula

  ✅ GET /api/stars/summary/<student_id>/<month_key>
     - Get complete star summary
     - Shows carry-in, awards, usage, available

  ✅ GET /api/stars/bonus/<student_id>/<date>/<month_key>
     - Calculate star bonus for a date
     - Applies +100 per normal use if score >= -50

  ✅ GET /api/stars/limits
     - Get all star system limits and rules
     - Daily, monthly, and usage limits

  ✅ GET /api/stars/calculation-formula
     - Get the unified calculation formula
     - Shows components and examples
